## 本项目仅供个人娱乐使用，请勿做其他用途
> 项目简介
> 实时或定时抓取各网络平台的热搜数据，基于weblogic爬虫框架一个可自定义、可扩展的抓取数据工程；
> 目前已实现，某乎、某音、某度、某博等平台的热搜数据抓取
